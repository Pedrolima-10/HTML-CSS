Trabalhando com listas

Listas ordenadas

Trabalhando com listas ordenadas usando tag {OL}, que deixar todo item que está entre {li} em ordem :

  1. Acordar
  2. Ligar para joão
  3. Tomar café
  4. Escrevar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Listas não ordenadas

Para trabalhar com listas não ordenadas usamos a tag {UL}, que segue o mesmo raciocinio mas os itens não são em ordem numerica como a outra lista :

  • Pão
  • Leite
  • Tomate
  • Alface
  • Arroz
  • Feijão

Listas misturadas

Para usar listas mistas usamos a tag {ol} dentro da tag {ol} assim fazendo 2 ou mais listas dentro da lista principal.

  1. Linguagens antigas
    1. Clipper
    2. Visual Basic
    3. Fortan
    4. Delphi
  2. Linguagens Novas
    1. Php
    2. Python
    3. javaScript
    4. Kotlin

Listas misturadas ol e ul

Podemos usar a mesma lógica usar uma {ul} dento de {ol} ou vice verça

  1. Nitendo
    • Mario Bross
      • Mario Bross 1
      • Mario Bross 2
    • Ninja Gaiden
  2. Xbox
    • Hallo
    • Forza
  3. PlayStaion
    • God of war
      • God of war 1
      • God of war 2
      • God of war 3
    • Read dead

Listas de definição

na lista definição usamos a tag {dl} e dentro dessa tag usamos a tag {dt} que é o termo, e dentro dessa tag usamos a tag {dd} que seria a escrição do termo :

JavaScript
Linguagem de progamação para criar a interatividade de um site.
Html
linguagem de marcação para a criação de conteudo de um site.
Css
linguagem de marcação para a criação do desing de um site.
