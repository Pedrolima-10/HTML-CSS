Reproduzindo audio

Vamos aprender a reproduzir áudios usando html
abixo está um podcast do professor gustavo guanabara, aperte play para reproduzir:

Para reproduzir áudios dessa maneira você precisa usar a tag {audio} seselcionar o áudio que vai querer, usar o comando {controls} para adicionar a barra de dar play e pausar, e por ultimo usar o comando {autoplay}, assim seu áudio estara disponivel no seu site como no exemplo acima.


Outra forma de adicionar áudio para navegadores que não tem compatibilidade com aulguns formatos de arquivos é assim:
{source src="midia/guanacast-33.mp3" type="audio/mpeg"}PARA mp3
{source src="midia/guanacast-33.ogg" type="audio/ogg"} PARA ogg
{source src="midia/guanacast-33.wav" type="audio/wav"} PARA wav

Lembrando o html vai testando esses códigos em ordem para ver qual vai funcionar, caso nenhum funcione, infelizmente seu navegador não é compativel com áudio clique aqui para baixar o mp3. você tem que se atentar no type que vai colocar pois cada arquivo é um type diferente

Lembrando que a tag {audio} tem alguns parametros para usar um deles é o {preload} que você escolhera o "metadata ou o none ", pois não é recomendado deixar como padrão. e você escolhera se vai querer o áudio com o {controls} que o usuário que apertara o play ou com o {autoplay} que o áudio começa automatico
