Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu própio servidor

Para ursamos vídeos em sites direto do seu proprio servidor, o modo simples de fazer isso é usando tag {video} e indicando qual vídeo você quer colocar no site, lembrando use o parâmetro {controls} para aparecer os botões de player e pause para o consumidor do site, para indicar uma capa para esse vídeo use o parâmetro {poster} e selecione a imagem que irá querer, o parâmetro {width} indica o tamanho de exibição do vídeo e da capa.
