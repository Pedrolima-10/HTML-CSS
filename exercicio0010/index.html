Trabalhando links externos

Para usarmos links externos dentro de um site precisamos envelopar o texto que queremos usar como âncora para um limk e usar a tag: {a} veja o exemplo:

Entre no canal do telegram do curso em video

Se inscreva no canal do curso em video

Para abrir um link exteno em outra janela e sem precisar fechar a janela do seu site, você precisa usar duas tags dentro da tag {a} são : {target="_blank"} e a tag {rel="external"}

Trabalhando com links em imagens

Para usamos links externos em imagens precisamos usar a mesma logica usada a cima, porem ao inves de envelopar texto você envelopa o código da imagem, exemplo abaixo com a logo do meu time de coração que clicando você ira para o site oficial do clube.

Trabalhando com links internos

Para usarmos links internos vamos precisar criar outro arquivo html na mesma pasta que está o principal, e assim usar mesma logica mas ao inves de usar a url de um site usaremos o diretorio da página criada, veja o exemplo a seguir:

Está é a primeira página, para ir para a proxima página clique aqui

Para identificar a proxima página você vai usar o {rel="next"} dentro da tag {a}, do mesmo jeito que usamos em Links externos, e para indentificar a página anterior usamos a tag {rel="prev"}.

Links para download

Para usar links para fazer download direto do seu navegador, você tem que reconhecer o que se trata o link, o formato dele pois cada formato tem seu tipo de código, [exemplo formato zip] : você envelopa o oque vai ser seu link e coloca dentro da tag {a} após isso você usa a tag { download="nome do seu arquivo.zip"} e depois disso a tag {type= "application/zip}

*Lembrando, cada formato tem seu código, para saber o código do formato que você quer acesse o site : iana.org
